A Stream Deck profile designed for RanchRP in RedM to enhanced streaming experiences and productivity.

## Prerequisites

Before using this profile, ensure you have the following installed:

- **Elgato Stream Deck Software** (latest version recommended)
  - Download from: [Elgato Stream Deck Software](https://www.elgato.com/en/downloads)
- **FXCommands Plugin** (required for this profile)
  - See installation instructions below

## FXCommands Plugin Installation

This profile requires the **FXCommands** plugin to function properly. Follow these steps to install it:

1. Download the FXCommands plugin from the [Github Release](https://github.com/josh-tf/fxcommands/releases/tag/v1.2.2.1)
2. Double-click the downloaded `.streamDeckPlugin` file
3. Follow the installation wizard
4. Restart the Stream Deck application

## Profile Installation

1. Ensure you have the Stream Deck software running
2. Locate the `RanchSDProfile.streamDeckProfile` file
3. Double-click the profile file, or
4. In Stream Deck, go to **Edit Profiles**, at the bottom of the profiles list, select the dropdown arrow **v** â†’ **Import Profile** and select the file
5. The profile will be imported and ready to use

## Usage

Once installed, select the **REDM** from your profile list in the Stream Deck application. The buttons configured in this profile will be available on your Stream Deck device.

## Features

This profile includes various actions and shortcuts configured for:
- Quick access to frequently used Emoji
- Frequent used Commands
- Clothing adjustment

## Troubleshooting

### Profile Not Loading
- Ensure Stream Deck software is up to date
- Verify that FXCommands plugin is installed and enabled
- Try restarting the Stream Deck application

### FXCommands Not Working
- Check that FXCommands plugin is installed correctly
- Verify the plugin is enabled in Stream Deck settings
- Reinstall FXCommands if issues persist
- Check the Stream Deck logs for error messages

### Buttons Not Responding
- Ensure all required plugins are installed
- Check that applications referenced in actions are installed
- Verify file paths in custom actions are correct
